# Backend — Sense Gardens API

Go 1.22+ REST/GraphQL API built with Fiber and gqlgen.

## Stack
- Go 1.22+
- Fiber (web framework)
- gqlgen (GraphQL)
- Flat GeoJSON files (no database)

## Folder Structure
```
backend/
├── cmd/server/        # entry point (main.go)
├── graph/             # GraphQL schema and resolvers
│   └── generated/     # auto-generated by gqlgen
├── data/              # GeoJSON data files
├── .env.example       # copy to .env and fill in keys
├── go.mod
└── go.sum
```

## Setup
```bash
cp .env.example .env
go mod tidy
go run cmd/server/main.go
```
