# Good First Tasks

This page lists beginner-friendly tasks for contributors who want to help improve the Sensible Living Foundation website and digital infrastructure.

These tasks are small, practical improvements that help the project move forward.

If you are new to the project or new to open-source contributions, these tasks are a great place to start.

Before starting, please review:

- CONTRIBUTOR_START_HERE.md
- WEBSITE_BLUEPRINT.md
- PROJECT_SCOPE.md

Then visit the Issues tab and comment on the issue you would like to work on.

---

# Task 1: Homepage Call-To-Action Review

Review the current homepage:

https://www.sensiblelivingfoundation.org

Look for places where visitors should be encouraged to take action.

Examples of calls to action:

• Donate  
• Volunteer  
• Join the newsletter  
• Partner with the foundation  

Deliverable:

Write a short document suggesting better locations and wording for calls to action.

Goal:

Help visitors quickly understand how they can support the foundation.

---

# Task 2: Improve Homepage Hero Message

The hero section is the first thing visitors see.

Visit the homepage and review the main message.

Goal:

Suggest a clearer and more compelling headline that communicates the foundation mission.

Example areas to improve:

• clarity of the mission
• emotional connection
• impact-focused language

Deliverable:

Propose 2–3 improved hero messages.

---

# Task 3: Basic SEO Page Review

Visit the website and review key pages.

Examples:

• homepage  
• financial literacy program page  
• Sense Gardens page  

Look for:

• page titles
• headings
• keyword opportunities
• missing descriptions

Deliverable:

Write a short list of SEO improvement suggestions.

Goal:

Improve search engine discoverability.

---

# Task 4: Newsletter Signup Ideas

Review the current website and identify where newsletter signup forms could appear.

Examples:

• homepage
• footer
• donation page
• program pages

Deliverable:

Write recommendations for where newsletter signup should appear and what the signup message should say.

Goal:

Increase the number of supporters who subscribe to updates.

---

# Task 5: Volunteer Signup Form Suggestions

Design a simple volunteer signup form structure.

Suggested fields:

• full name  
• email  
• phone number  
• area of interest  
• availability  

Volunteer categories may include:

• Sense Gardens  
• financial literacy education  
• community events  
• outreach and marketing  

Deliverable:

Create a recommended form layout and explain how it should work.

Goal:

Make it easier for community members to volunteer.

---

# Task 6: Partnership Page Suggestions

Suggest improvements for a corporate partnership page.

The page should help businesses understand how they can support the foundation.

Ideas may include:

• sponsorship opportunities  
• employee volunteer programs  
• partnership benefits  
• community impact

Deliverable:

Create a suggested outline for the partnership page.

---

# Task 7: Impact Story Ideas

Suggest ways the foundation could share stories about its mission.

Examples:

• founder story  
• community impact stories  
• future program goals  

Deliverable:

Write ideas for storytelling sections that could appear on the website.

Goal:

Help visitors emotionally connect with the mission.

---

# Contributor Tips

Good contributions should be:

• clear  
• practical  
• easy to implement  
• aligned with the foundation mission  

Remember that the foundation operates as a nonprofit organization, so recommendations should prioritize simplicity and sustainability.

Thank you for helping improve the Sensible Living Foundation’s ability to serve communities.
