PROJECT_SCOPE.md # Project Scope

## Project Name
Sensible Living Foundation Website Improvement and Automation Project

## Purpose
Improve the foundation website’s user experience, supporter flows, SEO, and automation.

## Objectives
1. Improve homepage clarity and calls to action
2. Create better donor, volunteer, and partner flows
3. Improve SEO and Google discoverability
4. Build newsletter and donor automation planning
5. Organize work for contributors on GitHub

## In Scope
- UX/UI audit
- SEO recommendations
- automation workflow planning
- form and intake flow planning
- content recommendations
- GitHub issue tracking

## Out of Scope for Phase 1
- full custom website rebuild
- advanced backend development
- full CRM migration
- paid ad management

## Phase 2 Deliverables (Interactive Map)
All code lives inside this repo under two folders:

- backend/   → Go + Fiber + GraphQL API serving GeoJSON data
- frontend/  → Next.js 14 + Mapbox interactive map

Delivered as a single Pull Request to the client's original repo.

## Phase 1 Deliverables
- UX/UI audit
- SEO audit
- automation plan
- contributor issue backlog
